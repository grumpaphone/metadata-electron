# Security Policy

## Reporting a Vulnerability

If you discover a security issue in metadata-electron, please report it
privately. Do not open a public GitHub issue for security-sensitive reports.

Email: wtendy@gmail.com

Please include:

- A description of the issue and its impact.
- Steps to reproduce, or a minimal proof-of-concept.
- The affected version / commit.

We will acknowledge receipt within a reasonable timeframe and work with you
on a coordinated disclosure.

## Supported Versions

Only the latest release on `main` is actively supported with security fixes.
